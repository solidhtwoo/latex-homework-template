latex-homework-template
=======================

这是一份我自用的LaTeX作业模板, 修改和汉化自
https://github.com/jdavis/latex-homework-template

## 特性

1. 标题页
2. 代码高亮
3. 可调的题号
4. 常用数学宏

需要安装xelatex,Python3以及pygments包

```latex

xelatex -shell-escape homework.tex

```
